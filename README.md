# Currency Converter

**Автор:** Аделя Рашитова  
**Дата:** 6.05.2026

## Описание

Приложение для конвертации валют с использованием внешнего API и сохранением истории операций. Позволяет быстро и удобно конвертировать суммы между популярными валютами с актуальными курсами.

## Функциональность

* Конвертация между популярными валютами (USD, EUR, GBP, JPY, CNY, RUB, CAD, AUD, CHF).
* Получение актуальных курсов через API [exchangerate-api.com](https://www.exchangerate-api.com/).
* Кэширование курсов на 1 час для снижения числа запросов к API.
* Сохранение истории конвертаций в файл `history.json`.
* Отображение истории в табличном виде с возможностью очистки.
* Валидация ввода: проверка на число и положительность суммы.
* Интуитивно понятный графический интерфейс (GUI).
